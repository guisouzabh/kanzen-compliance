# Vanttagem LGPD Landing

Landing publica em React + Ant Design para divulgacao do sistema Vanttagem LGPD.

## Rodar local

```bash
npm install
npm run dev
```

A aplicacao sobe em `http://localhost:5175`.

## Configuracao

Crie um `.env` baseado em `.env.example`:

```bash
VITE_WHATSAPP_NUMBER=5511999999999
```

## Build

```bash
npm run build
npm run preview
```
